# Inventory Web (React + Tailwind) — Arabic default

هذا مشروع واجهة ويب لجرد أصناف الموردين. النسخة مهيأة لتعمل على GitHub Pages تحت المسار `/inventory-web/`.

## مميزات
- تحميل ملفات Excel (.xlsx, .xls, .csv) أو PDF واستخراج الأصناف (محليًا في المتصفح).
- جرد باستخدام: كتابة الكود، سكان بالباركود (مستقبليًا باستخدام @zxing/browser)، أو البحث بالاسم.
- لغة افتراضية: **العربية** مع زر للتبديل إلى الإنجليزية.
- تصدير النتائج إلى ملف Excel.

## تشغيل محليًا
1. ثبت Node.js (v16+) وnpm.
2. نفّذ:
```bash
npm install
npm run dev
```
3. افتح `http://localhost:5173`.

## نشر على GitHub Pages
1. ارفع المجلد إلى ريبو باسم `inventory-web` في حسابك `hema-tech1`.
2. شغّل `npm run build`.
3. يمكنك استخدام gh-pages لنشر مجلد `dist`:
```bash
npm install -g gh-pages
npm run build
npm run deploy
```
أو من إعدادات الريبو → Pages → اختر فرع `gh-pages` أو `main` و`/ (root)` بناءً على إعدادك.

## ملاحظات
- معالجة PDF تعتمد على pdf.js ولن تكون مثالية لجميع أنواع PDF، لكن تعمل جيدًا مع ملفات نصية ومنسقة.
- سكان الباركود يحتاج إضافة واجهة كاميرا (موجودة كاعتماد @zxing/browser) — في هذه النسخة زر السكان يظهر لكن يتطلب تنفيذ مسار سكان منفصل.
